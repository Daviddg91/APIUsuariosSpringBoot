Aplicación y API Java

Este proyecto es una aplicación web y una API Rest desarrollada con Spring Boot, tiene una interfaz de gestion de Clientes donde se puede acceder a las funcionalidades CRUD con validación customizada, búsqueda, paginación, subida de ficheros y ordenación de registros referente a los clientes.
También tiene una interfaz de productos donde se puede elegir entre los clientes y los productos para hacer los pedidos, luego estos pedidos se guardan automaticamente en una tabla.
En su panel de administración logueandose con el usuario por defecto con rol de ADMIN, se puede acceder a una interfaz de administracion donde se pueden crear, asignar y modificar los usuarios y los roles.
También he podido experimentar con Beans de java donde se desarrollan funcionalidades especificas en las que se realizan páginas como Tecnologías aleatorias, donde se muestra una página diferente de forma aleatoria cada vez que se actualiza el navegador, y la pagina Saludo, que muestra una página distinta dependiendo de la hora del día.

Esta aplicación incluye tecnologías como Spring Security, Spring Data, JPA (creación de los modelos de bases de datos automático mediante anotaciones de JPA), Maven, renderizado con Thymeleaf, una librería que se incorpora a Spring boot con gestión de plantillas . Ajax, Jquery, Bootstrap y diseño Front-end con menú reponsivo.
