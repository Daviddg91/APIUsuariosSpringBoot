Aplicación y API Java

Este proyecto es una aplicación web y una API Rest desarrollada con Spring Boot, tiene una interfaz de gestion de Clientes donde se puede acceder a las siguientes funcionalidades CRUD:
  • Formularios con validación customizada
  • Búsqueda, paginación, y ordenación de registros en las tablas
  • Subida de ficheros
  • Interfaz de administración de Usuarios y Roles disponible para usuarios logueados como administradores
  • Interfaz de pedidos donde se pueden realizar pedidos asignando un cliente a una lista de productos y realizar un pedido.

Las paginas que explotan las funcionalidades de los Beans son las siguientes:
  • Tecnologías aleatorias, donde se muestra una página diferente de forma aleatoria cada vez que se actualiza la página
  • Saludo, que muestra una página distinta dependiendo de la hora del día

Esta aplicación las siguientes incluye tecnologías:
  • API Rest
  • Spring Security
  • Spring Data contra MySQL
  • JPA (creación automática de los modelos de bases de datos mediante anotaciones)
  • Maven
  • Renderizado con Thymeleaf
  • Ajax
  • Jquery
  • Bootstrap
  • Diseño Front-end con menú reponsivo
  • Beans customizados para realizar diversas funcionalidades
